### Installation

```bash
# Clone repository
git clone [your-repo-url]
cd reflexa-ai-chrome-extension

# Install dependencies
npm install

# Build extension
npm run build

# Load in Chrome
# 1. Go to chrome://extensions/
# 2. Enable "Developer mode"
# 3. Click "Load unpacked"
# 4. Select the `dist` folder
```
