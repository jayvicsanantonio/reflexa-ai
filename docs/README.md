## ğŸ” What is Reflexa AI?

Reflexa AI is a wellness-centered Chrome Extension that transforms everyday reading into calm, reflective micro-sessions with AI-powered insights.

### Key Features

- **AI-Powered Summaries** - Get three-bullet summaries (Insight, Surprise, Apply) using Chrome's Gemini Nano
- **Guided Reflection** - Thoughtful questions to deepen your understanding
- **Privacy-First** - All AI processing happens on your device - no data sent to servers
- **Calming Experience** - Zen-inspired design with breathing animations
- **Accessible** - Full keyboard navigation and screen reader support
