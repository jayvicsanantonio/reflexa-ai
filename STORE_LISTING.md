# Reflexa AI – Chrome Web Store Listing

## Short Description
Calm, on‑device reflections for better focus. Summaries, prompts, and gentle guidance powered by Chrome’s built‑in Gemini Nano.

## Detailed Description
Reflexa AI helps you pause and reflect while reading online. It runs entirely on your device using Chrome Built‑in AI APIs (Gemini Nano) — no accounts, no servers, no data collection.

- On‑device summarization in your language
- Two thoughtful questions to deepen understanding
- Optional proofreading, translation, and tone adjustments
- Accessible UI: keyboard navigation, reduced‑motion, visible focus
- Privacy‑first: reflections stay on your device; export anytime

## Key Features
- In‑page overlay with gentle animations (respecting reduced motion)
- AI capability detection and graceful fallbacks
- Local storage of reflections with export/delete

## Permissions Rationale
- Site access: to render the overlay and extract on‑page text for on‑device summarization
- Storage: to save your reflections and settings locally
- Active Tab + Scripting: to communicate with the active page (no background crawling)

## Data Disclosure
- Data collected: None
- Data shared: None
- Data used for personalization: No
- Account/Sign‑in required: No
